Repository
--

This repository contains the source code of the Loci Lamp assembly manual. Loci Lamp is an open-source laser cut lamp provided by Tchibo, New Production Institute and InMachines. 

In this assembly manual you will find instructions on how to produce and assemble the Loci Lamp, as well as the cutting files and BOM.
